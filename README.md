# info about Enzo's 

## 基础4属性：Elem4

### 攻击力

* 剑类：1000
* 短剑类：800

### 攻速

* 0～100：需要长时间蓄力或咏唱使用，无法连续使用；
* 101～200：需要蓄力或挥舞使用，有一定的间歇期，无法连击；
* 201～300：需要准备使用，可以缓慢的连击；
* 301～400：可以直接使用，并可以连击；
* 401～500：几乎瞬发，可以造成迅速的连击；
* 501～600：一次甚至可以造成多重伤害；

### 范围

* 0～100：贴身攻击；

* 101～200：近距离交战；

* 201～300：较长的近距离战斗；

* 301～400：短途远程攻击；

* 401～500：远程攻击；

* 501～600：视距外攻击；

### 重量

* 0：无物理重量
* 1：纸级
* 2：暗器级
* 3：匕首级
* 4：短剑级
* 5：佩剑级
* 6：重剑级
* 7：重型武器级
* 8：重型器械级
* 9：规格外超重级

## 词条：Item

* 技能类`▸`：0真名/1高稀有度/7附魔
> 技能名：技能描述

* 常时效果类`◆`：2最终强化/6共鸣
> 技能名：技能描述（可以有“攻击力 + 500”这样的简单数值描述）

* 属性加成类`▹ `：3魔强/8固有
> 属性名 +加成数值 

* 镶嵌类`● `：4,5镶嵌
> 宝石名：属性名 +加成数值
